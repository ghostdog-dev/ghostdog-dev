## Repository Guide

To make it easier to browse, every repository follows a naming convention. Use these prefixes as search keywords on my GitHub profile:

| Prefix | Meaning |
|--------|---------|
| **`app-*`** | Personal & freelance projects — real-world applications |
| **`openclassrooms-dev-app-python-*`** | Coursework — OpenClassrooms *Développeur d'application Python* track |
| **`openclassrooms-dev-app-react-*`** | Coursework — OpenClassrooms *Développeur d'application JS / React* track |
| **`learning-python-*`** | Personal Python projects from before any formal training |

> *Quick tip:* type one of these prefixes into the repo search box on this profile to filter instantly.
